Please follow the below steps to establish a connection between QC ALM with the automation suite:
-------------------------------------------------------------------------------------------------
1. Download and Run TDConnector.exe file from HP Add-ins page.(Help-->Add-ins Page-->HP Quality Center Connectivity-->Download Add-in)
2. Make sure the machine has 32 bit version of JDK and JRE
3. Execute the batch file alm.bat from the project folder batchfiles. 
   You will see a success message as like the given file Alm_Service_Success_Msg. Click OK on it.
